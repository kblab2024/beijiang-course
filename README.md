# 北疆景物與相關歌曲 — 8 週深度文化 × 音樂課程（網站）

可瀏覽的課程網站，發布於 GitHub Pages：

➡️ **https://kblab2024.github.io/beijiang-course/**

- 8 週、每週深講**一首主打歌、一位主角**（八藝人八歌），從該地區的**深度文化**切入（地名由來・傳說・歷史・文物・民族音樂）。
- 由各週 `slides.md`／`script.md` 原稿**自動產生**（產生器：`build_site.py`）。
- 每張投影片＝一個區塊（標題・條列・圖片・YouTube 嵌入）；**講者稿／講義補充**收在可展開區；每週附 **PDF 下載**。

## 結構
- `docs/` — GitHub Pages 網站根目錄（`index.html`、`week1..week8.html`、`site.css`、`assets/images/`、`downloads/`）。

> 原稿為單一事實來源；本網站為 `/docs` 靜態輸出。繁體中文；史實附年代、傳說標「相傳」、不確定標「待查證」；歌曲段只取意象、不引整首歌詞。
